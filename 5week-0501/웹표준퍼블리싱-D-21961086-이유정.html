1. 테이블 요소 - table, tr, th, td

  • 선언 <table>

2. 셀 병합 - rowspan, colspan

  • 가로로 합치기 colspan
  • 세로로 합치기 rowspan

3. 제목 및 요약문 - caption, summary

  • Summary
    - 간단한 제목보다 테이블의 내용을 설명
    - 시각적으로 테이블 접근이 쉽지 않은 사용자를 위한 배려
  • Caption
    - 테이블의 내용을 대표할 수 있는 제목을 작성

4. 행 그룹화 요소– thead, tfoot, tbody

  • tbody: 논리적 구분을 위해 여러번 등장할 수도 있다

5. 제목 셀과 내용 셀의 연관성– scope

폼 관련 요소

1. 폼의 정의와 요소:

  • 웹 문서가 서로 상호작용을 할 수 있도록 하는 역할
    - 입력 상자, 라디오 버튼, 체크 박스, 목록상자(combo box), 전송버튼
  • 폼 요소
    - action = "URL" : 폼의 내용을 처리하기 위한 서버의 URL
    - method="get" : 웹 브라우저의주소 창에 변수 값이 노출. 짧을 경우. 빠름
    - method="post" : 긴 경우. 보안 문제 때문에 데이터값을 노출시킬 수 없을 때

2. 그룹화, 제목

  • 그룹화, 제목
    - fieldset : 여러 개의 폼 요소를 그룹화 하여 좀 더 구조적으로 표현 가능.
    - legend : 어떠한 성격의 콘텐츠인지 제목 형식으로 알려줌. 한 번만 작성 가능.

3. 레이블

  • 각 폼 컨트롤의 연관 관계와 설명을 추가하는 역할
    - 입력 폼과 1:1로 대응
    - 사용하기 어려울 경우 input에 title 속성으로 대체 가능
  • 명시적 레이블(권장)
  • 암묵적 레이블

4. input

  • Type

CSS의 속성은 기본적으로 하위 요소로 상속되는 속성과 상속되지 않는 속성이 있습니다.
