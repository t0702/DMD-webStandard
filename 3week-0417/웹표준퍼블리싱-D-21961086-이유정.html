img 태그

<img/>태그를 사용하여 이미지를 넣을 수 있습니다.
src를 통해 경로를 입력해줍니다. width의 값으로 이미지의 가로사이즈를 지정하며 height의 값으로 세로사이즈를 지정합니다.
alt란 이미지의 묘사 내용을 말합니다. title은 이미지의 제목을 말하며 이미지에 마우스를 올리면 title이 표시됩니다.

절대경로를 사용한 예입니다.

상대경로를 사용한 예입니다.
소스파일을 함께 첨부하지 않으면 사진이 표시되지 않습니다.

a 태그

a태그의 href 으로 이동할 주소를 지정합니다.
target 속성은 _blank 값으로 새 창을 띄우도록 지정합니다.

< a href="http://www.kaywon.ac.kr/" target="_blank" alt="계원예술대학교 바로가기" title="타이틀 값입니다.">계원예술대학교 바로가기</a>

테이블 관련요소

1. 테이블 요소 - table, tr, th, td

  • 선언 <table>
  • 행 <tr>
    제목 셀 <th>
    내용 셀 <td>

2. 셀 병합 - rowspan, colspan

  • 가로로 합치기 colspan
  • 세로로 합치기 rowspan

3. 제목 및 요약문 - caption, summary

  • Summary
    간단한 제목보다 테이블의 내용을 설명
    시각적으로 테이블 접근이 쉽지 않은 사용자를 위한 배려
  • Caption
    테이블의 내용을 대표할 수 있는 제목을 작성

4. 행 그룹화 요소– thead, tfoot, tbody

  • 선언순서 : thead, tfoot, tbody
  • thead
    테이블의 헤더 영역. 반드시 th요소를 사용하여 제목 셀이라는 것을 명시
  • tfoot
    위치에 관계 없이 테이블의 가장 마지막 행에 출력
  • tbody
    논리적 구분을 위해 여러번 등장할 수도 있다

5. 제목 셀과 내용 셀의 연관성– scope

  • scope
    scope=“row”행의 제목
    scope="col"열의 제목

폼 관련 요소

1. 폼의 정의와 요소:

  • 웹 문서가 서로 상호작용을 할 수 있도록 하는 역할
    입력 상자, 라디오 버튼, 체크 박스, 목록상자(combo box), 전송버튼
  • 폼 요소<
    action = "URL" : 폼의 내용을 처리하기 위한 서버의 URL
    method="get" : 웹 브라우저의주소 창에 변수 값이 노출. 짧을 경우. 빠름
    method="post" : 긴 경우. 보안 문제 때문에 데이터값을 노출시킬 수 없을 때

2. 그룹화, 제목

  • 그룹화, 제목
    fieldset : 여러 개의 폼 요소를 그룹화 하여 좀 더 구조적으로 표현 가능.
    legend : 어떠한 성격의 콘텐츠인지 제목 형식으로 알려줌. 한 번만 작성 가능.

3. 레이블

  • 각 폼 컨트롤의 연관 관계와 설명을 추가하는 역할
    입력 폼과 1:1로 대응
    사용하기 어려울 경우 input에 title 속성으로 대체 가능
  • 명시적 레이블(권장)
  • 암묵적 레이블

4. input

  • Type
    text : 한 줄 글 상자
    password : 비밀번호 입력. *나 ●로 표시
    radio : 여러 개의 라디오 버튼 중 하나만 선택 가능 (같은 name명 끼리 묶음)
    checkbox : 다중 선택이 가능한 체크 박스
    file : 파일을 첨부 형태로 서버로 보낼 때 사용하는 컨트롤

5. select, option, optgroup

6. textarea
